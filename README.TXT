README:

The keys used in this game are:

Arrow keys.....................................Move you around
, and .   .....................................Set the game's speed (<,>)
+         .....................................Shows the current CCs you have
q         .....................................Quits the game
F2        .....................................Saves the game
S         .....................................Sound on/off

-Type RDI to read the instructions.
-Beat every level to get the special cheat code.
-You may see "file not found" messages while using the menu. Don't worry
 about these.


-Note:
  This program was made partly in qb45 and qb71. It needs at least a 386/486(*sigh*), a VGA card, and a PC speaker. <- Finding these shouldn't be much of a problem.
